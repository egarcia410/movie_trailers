# Movie Trailer Project

Displays the top 6 movies of all time! The movies are showcased with their poster artwork and once clicked, displays the movies' trailer. 

## Installation 

```
Open Terminal
	$ git clone https://github.com/egarcia410/movie_trailers.git
Navigate to Project Folder
	$ cd movies
Run Program
	$ python entertainment.py
Enjoy
```
